## 整体需求说明
本项目是一个类似苹果Reminder的功能，底层代码使用Markdown格式，但是会在Markdown格式的基础上进行修改。

1. 用户启动软件后，首先需要在本地选择一个目录，该目录作为Reminder的工作目录，这样做的目的是便于Reminder的多端同步和备份
2. 在系统设置中添加一个设置项，叫做工作目录。用户可以修改工作目录，并读取该工作目录下的Reminder列表
3. 工作目录下保存多个TXT文件，每个TXT文件对应一个待办事项列表
4. 在工作目录下创建配置文件config.yaml，用于保存系统的所有配置

## 文件解析说明
每一个任务列表的底层是一个TXT文本，每个任务被分为上下两行，上行为JSON，记录该任务的相关信息，下行为text，记录该任务的具体内容。
```json
{
    "CreateTime": "2026-01-01 12:30:00",
    "Deadline": "2026-01-01 12:30:00",
    "Level": 1,
    "Status": "TODO",
}
```
| 字段 | 数据类型 | 说明 |
| ------- | ------- | ------- |
| CreateTime  | Time stamp | 该事项被记录的时间，用户输入时由系统自动创建  |
| Deadline  | Time stamp  | 该事件的截止时间  |
| Level | Int | 大于等于一的整数，Level为一表示该事件为独立事件，Level为二表示该事件为上一个Level为1的事件的子事件，以此类推 |
| Status | 字符串枚举值 | 枚举值包括TODO（待处理）、DONE（已完成）和CANCELED（已取消） |


## 设置
### 窗口整体设计
1. 窗口标题为“Settings"
2. Setting内容包括:
   1. 第一个section名称为"通用（英文为General）":
      1. 第一个选项为语言，右侧为下拉框，选项包括`中文`和`English`，无论系统是什么语言，这里都显示`中文`和`English`
      2. 第二个选项为外观，右侧为下拉框，选项由上而下依次为`浅色`、`深色`、`跟随系统`，英文分别为`Light`、 `Dark`和`System`
      3. 第三个选项是`工作目录`，在工作目录的右方显示当前工作目录,并在下方显示四个按钮,分别是:
         1. `修改工作目录`: 点击后弹出Folder，让用户可以选择新的工作目录，用户选择后，如果用户已经有工作目录，需要进行修改，则需弹出二次确认框，左侧选项是`放弃`，右侧选项是`确认修改`，右侧选项为红色字体
         2. `打开`：点击后在Folder中打开工作目录
         3. `config.yaml`：点击后使用系统中的默认文本编辑器打开config.yaml
         4. `刷新`：系统会重新读取工作目录下的文件
   2. 第二个section名称为`优先级`，该部分会显示系统中内置的以及用户自定义的优先级，每个优先级从左到右依次显示以下部分：
      1. 最左侧是一个圆形的色块，说明该优先级对应的颜色，用户点击后会显示Color Picker，提供了三十种默认颜色以及用户自己选定颜色的方案
      2. 色块的右边是该优先级的名称，该名称的左侧与色块对齐, 用户点击后可以给优先级修改名称，并且可以使用Ctrl Z撤销修改动作，系统中的内置优先级不支持修改名称（这里的颜色显示为百分之八十灰度在暗色模式下，需要进行对应的调整，以提供更好的显示效果）
      3. 在名称的右边是三个选项，分别是`粗体`、`下划线`和`斜体`，用来表示该优先级的文字显示形式
      4. 最右边则显示垃圾桶图标。系统内置的优先级不显示垃圾桶图标，用户自定义的优先级显示垃圾桶图标。默认情况下，垃圾桶图标为灰色，当用户鼠标hover时显示为红色，用户点击删除需要进行二次确认
      5. 在所有的优先级下方显示添加优先级按钮,用户点击后会新增一个优先级,该优先级的配置与系统中最下方的优先级保持一致,除了名称为`自定义优先级`之外,用户点击自定义优先级后，鼠标会自动focus到优先级的名称输入框并全选默认名称，以方便用户输入自定义名称,该名称不能为空，如果为空的话，则会提示用户名称不能为空
   3. 第三个section名为`番茄时间`, 该部分会显示系统中内置的以及用户自定义的番茄时间。每个番茄时间从左到右依次显示以下部分:
      1. 一个表示番茄时间的图标
      2. 番茄时间的时长，格式为 `__ 小时 __ 分钟`，用户点击后，会弹出数字框，让用户选择时长，默认值为零小时三十分钟
      3. 最右边是垃圾桶图标，系统内置的时长不显示垃圾桶图标，用户自定义的时长显示灰色垃圾桶图标，鼠标hover时显示为红色，删除时需要二次确认
      4. 所有番茄时长的下方显示添加按钮，用户点击后会新增一个番茄时长，时长默认时间是零小时三十分钟
